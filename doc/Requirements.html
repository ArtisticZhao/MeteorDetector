必要なソフトウェア

MeteorDetector は Windows (x64) で動作します.
MeteorDetector の動作に必要なソフトウェアは以下のとおりです.
  • Visual Studio 2017 (Community Edition 以上) (Required)
    C++ コンパイラです.
    POCO C++ Libraries のライブラリを作成する場合に使用します.
    MeteorDetector の解析エンジンを開発する場合に使用します.

  • POCO C++ Libraries (Required)
    様々なプラットフォームに対応した C++ クラスライブラリです.

  • OpenCV library (Required)
    画像処理, 画像解析を行うライブラリです.

  • Visual Studio 2017 の Microsoft Visual C++ 再頒布可能パッケージ (Selectable)
    Visual Studio 2017 がインストールされていないコンピュータで MeteorDetector を実行する場合に必要です.
