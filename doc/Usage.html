使用方法

コマンドプロンプトから MeteorDetector を実行します.

例:

C:\uzanka\MeteorDetector\bin> MeteorDetector.exe /target=sample.avi


結果ファイルが data ディレクトリに作成されます。

コマンドラインオプション

使用できるコマンドラインオプションは以下のとおりです.
省略形も使用できます.

コマンドラインオプション 説明
/target=<video_path>
/t=<video_path> 		必須オプションです.
対象の動画ファイルを指定します.
使用できる動画ファイルの種類は, OpenCV に依存します.
/help
/h 		使用方法を表示します.
